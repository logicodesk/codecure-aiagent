Upgrade 1 — Real Model Training Pipeline

1
Load Tox21 with RDKit Descriptors

Parse SMILES from train/test SDF files. Compute 200 RDKit Morgan fingerprints (radius=2, 2048 bits) + 12 Lipinski descriptors per compound. Handle missing values with median imputation per task.

2
Train Ensemble Per Tox21 Task

For each of the 12 assay targets: fit XGBoost + Random Forest + SVM. Ensemble via soft-voting. Use stratified k-fold (k=5) to handle class imbalance. Save per-task ROC-AUC and PR-AUC.

3
SHAP on XGBoost per Task

Compute SHAP TreeExplainer values. Extract top-10 features per prediction. Map fingerprint bits back to SMARTS substructures for human-readable explanations. This feeds SHAPExplanation.jsx directly.

4
Serialize + Serve via FastAPI

Pickle 12 ensemble models. Load once at startup. /predict endpoint runs in under 500ms per compound. Return SHAP values, predictions, and bioisostere suggestions in one response payload.

Upgrade 2 — Batch Analysis Dashboard

CSV Upload + Population View

Add a "Batch Mode" tab. User uploads a CSV of SMILES. Backend processes all in parallel (asyncio + process pool). Returns: per-compound risk scores, interactive scatter plot (LogP vs MW colored by toxicity risk), correlation heatmap of the 12 Tox21 tasks, and a downloadable report CSV.

POST /batch-predict Recharts scatter plot Correlation heatmap Export to CSV

Upgrade 3 — Chemical Space Explorer (ZINC250k)

t-SNE / UMAP Chemical Space Map

Sample 5,000 compounds from ZINC250k. Compute Morgan fingerprints. Run UMAP dimensionality reduction. Color points by predicted toxicity risk. Let users click a point to see its structure and prediction. This is visually stunning and scientifically rigorous — a guaranteed crowd-pleaser.

Upgrade 4 — Comparative Analysis Mode

Side-by-Side Molecule Comparison

Allow users to compare a toxic compound against its bioisosteric swap suggestion side by side. Show delta scores for each Tox21 task, overlaid atom saliency maps, and a radar chart of property changes. This makes the Bioisosteric Swap Engine visually compelling rather than just text output.

It runs the Bioisosteric Swap Engine and shows the safer analog side-by-side with a before/after toxicity radar chart. This is the single most demo-able, judge-memorable feature. No other team will have automated medicinal chemistry in their UI.

Atom-Level Danger Highlighting on 3D Structure

The 3D MolViewer shows a ball-and-stick model. Overlay atom saliency scores as a heat-mapped color gradient — red atoms are the dangerous ones. Users literally see where the toxicity lives in the molecule. Combine this with a "click an atom to learn why it's dangerous" tooltip. Visually jaw-dropping.

Organ Risk Radar Chart

ToxKG maps toxicity to liver, heart, kidney, brain. Visualize this as a radar/spider chart with 5 axes (Hepato, Cardio, Nephro, Neuro, Pulmonary). Real drugs show characteristic patterns — paracetamol shows high hepatotoxicity, cardiotoxic drugs light up the heart. This is scientifically meaningful and visually beautiful.

Your VoiceAssistant already exists. Demo it live: speak "Predict toxicity of aspirin" in Hindi or Spanish, watch the multilingual transcription, fuzzy correction, and instant 3D structure appear. No competitor will demo AI in 5 languages. The judges will remember this moment.
